Machine Description

The DRL-344 is the flagship of high speed production drilling. Strength and speed come together in the DRL-344 to deliver unmatched productivity. Equipped with three heavy duty 20 hp spindles, this machine uses high performance spade and carbide tipped drills to good advantage, boosting your tons per hour. There is no waiting. Independent movement of the spindles in all axes allows each spindle to immediately move to its next hole without waiting for the other spindles to finish.

Machine Features

• High-speed carbide drilling
• Simultaneous multi-axis drilling
• 20 hp variable speed spindles
• 200-1,200 variable spindle rpm
• 24" independent spindle length travel
• Electric AC servo / ball screw drill feed system
• Automatic web location gaging and web depth hole compensation
• Thru-spindle coolant system
• #4 morse taper
• Positioning accuracy +/- 1/32"
• Machine weight approx. 38,000 lbs
• Air conditioned Windows XP controller
• Easily networked

Material Capacity

• Maximum material width: 44"
• Minimum material width: 4"
• Maximum material weight: 1,000 lbs/ft.
• Maximum hole size:
          - 1-7/8" with 15hp motors
          - 2-1/4" with 20hp motors
• Maximum flange height: 18"

Machine Description

The DRL-348TC is a high speed production drill line with a fully automated 5 position tool changer for each drill spindle.

This machine has been designed to reduce operator error during production by allowing the machine to make any necessary changes in tooling size depending on the part being produced. With a maximum material width capacity of 48" combined with the 1000 lb/ft conveyor system, the DRL-348TC is able to process anything from the smallest to the largest of structural members.

Machine Features

• Fully automated 5-tool changer on each spindle axis
• High performance carbide, spade, or twist bit drilling
• Simultaneous multi-axis drilling
• 20 hp variable speed spindles
• 100-2,130 variable spindle rpm
• Brushless AC servo / ball screw drill feed system
• Proportional valve controlled hydraulic positioning system
• Automatic web location gaging and web depth hole   compensation
• Independent web probing for flange hole operations
• Thru-spindle coolant system
• CAT 40 drill spindle taper
• Positioning accuracy +/- 1/32"
• Machine weight approx. 28,000 lbs
• Air conditioned Windows XP controller
• Controlled Automation's SICAM operating software®
• Easily networked

Material Capacity

• Maximum material width: 48"
• Minimum width: 4"
• Maximum material weight: 1,000 lb/ft
• Maximum hole size: 2-1/4"
• Maximum flange height: 18"
